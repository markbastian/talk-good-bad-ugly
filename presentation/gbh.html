Integrated Development Environment (IDE)

  • Arguably a developer's best friend
  • Auto-completion
  • Project management and tasking
  • Various executions and tasks
  • How do they support different languages?

IDEs: Eclipse

  • First one I used
  • Arguably the most popular free IDE
  • RCP is reviled
  • Excellent Java support, but no implicit Maven support
  • Early Scala plugin was very, very slow and memory intensive
  • False Scala errors
  • Current Scala and Clojure (Counterclockwise) plugins are good

IDEs: NetBeans

  • Native pom support
  • Best RCP platform
  • False errors when invoking Scala
  • Poorly supported Scala plugin
  • No working Clojure plugin for current version of NB

IDEs: IntelliJ IDEA

  • Awesome support for Java, Scala, and Clojure
  • Understands build.xml, pom.xml, build.sbt, project.clj
  • Java out of the box
  • Top-notch Scala plugin
  • Excellent free Clojure plugin (https://cursiveclojure.com)
