The Object Oriented Approach

  • The domain is represented by objects
  • Top-down decomposition
    • Is-a
    • Has-a
  • Object hierarchies are interdependent an non-separable

The Clojure Approach

  • The domain is represented by data
  • This is relatively unique to Clojure
  • Other FPs (e.g Scala) are still class driven
